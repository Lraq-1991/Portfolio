3.1 Null values in numerical columns¶

Will replace the missing value with one of the following strategies:

  • Replace it with a constant value. This can be a good approach when used in discussion with the domain expert for the data we are dealing with.
  • Replace it with the mean or median. This is a decent approach when the data size is small—but it does add bias.
  • Replace it with values by using information from other columns.

For the cases presented, we will replace nulls with values by using information from other columns.

In [243]:
# Create function to fill NaN cells with numeric values using interpolation

# Will use 'Quantity', 'Price Per Unit' and 'Total Spent' for the interpolation

# Formula: df['A'] * df['B'] = df['C']

def fill_values(row):
    
    try:
                
        if pd.isna(row.iloc[0]) and not pd.isna(row.iloc[1]) and not pd.isna(row.iloc[2]):
            row.iloc[0] = row.iloc[2] / row.iloc[1]
        elif pd.isna(row.iloc[1]) and not pd.isna(row.iloc[2]) and not pd.isna(row.iloc[0]):
            row.iloc[1] = row.iloc[2] / row.iloc[0]
        elif pd.isna(row.iloc[2]) and not pd.isna(row.iloc[1]) and not pd.isna(row.iloc[0]):
            row.iloc[2] = row.iloc[0] * row.iloc[1]
        else:
            return row
        return row 
        
    except Exception as e:
        
        print(e)
